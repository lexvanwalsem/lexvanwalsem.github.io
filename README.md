# Lex van Walsem | Full-stack Developer Portfolio

Dit is de broncode van mijn persoonlijke portfolio, live te zien op [lexvanwalsem.github.io](https://lexvanwalsem.github.io). 

Het ontwerp is gebaseerd op een **Brutalistisch & High-tech** concept, met een focus op performance en technische esthetiek.

## 🛠 Tech Stack

* **Framework:** [React](https://reactjs.dev/) + [Vite](https://vitejs.dev/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Animaties:** [Framer Motion](https://www.framer.com/motion/) & [ReactBits](https://reactbits.dev/)
* **Graphics:** [ShaderGradient](https://www.shadergradient.co/) (Canvas-based)
* **Runtime:** [Bun](https://bun.sh/)
* **Deployment:** GitHub Actions & GitHub Pages

## 🚀 Kenmerken

* **ASCII Hero:** Dynamische hero-sectie met gestaffelde regelanimaties.
* **Live Status Bar:** Technische statusweergave inclusief locatie, uptime en huidige tech-stack.
* **Bento Grid Projects:** Overzicht van mijn werk bij o.a. **Bold Digital** en freelance projecten.
* **Business Integration:** Sectie gewijd aan mijn **Minor Finance** en bestuurservaring bij de **Valley Vultures**.

## 📦 Ontwikkeling

Dit project maakt gebruik van **Bun** voor een snelle development cycle.

```bash
# Dependencies installeren
bun install

# Development server starten
bun run dev

# Productie build genereren
bun run build
